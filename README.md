## What’s inside
- `notebooks/figures/ai_society_figures.ipynb` — single clean notebook (7 cells).
- `scripts/repro.py` — generates figures/data.
- `figures/` — outputs created at runtime.
- `appendix/` — F,G materials (EN/RU).
